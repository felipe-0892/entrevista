Colinhas

Para usar margem ou padding no Bootstrap, é possível:
Usar r para usar margin-right ou padding-right
Usar x para usar margem ou padding na esquerda e direita
Usar y para usar margem ou padding na parte superior e inferior
Deixar o campo em branco para usar margem ou padding em todos os quatro lados do elemento
